1. 어제 만들엇던 슬라이드르 안보고 처음부터 끝까지 만들어보기 지문 1. 6개의 아이템을 가지고있는 슬라이드를 마크업한다. 2. 6개의 아이템에는 각각의 버튼과 div 화면이 있다. 3. 슬라이드 베너를 컨트롤할 수 있는 버튼 next 와 prev 만든다. 4. 어제 디자인을한 형식으로 스타일 시트를 구성한다. 1. next 버튼에 클릭 이벤트를 걸어 각각의 아이템의 슬라이드로 이동할 수 있게한다. 2. next 버튼을 클릭 하면 현재 보이는 아이템 div는 왼쪽으로 이동을하며, 다음이미지는 오른쪽에 준비되어 화면에 보일수 있는 위치에 온다. 3. next 버튼 클릭시 아이템 갯수 이상 클릭하였을 경우 처음 이미지가 나오도록 한다 -- prev 버튼 동일 반대 애니메이션 진행 1. 6개의 각각의 아이템의 버튼을 클릭했을때 버튼 인덱스에 맞는 div 화면이 슬라이딩 되어 나오도록 한다. 2. 슬라이드가 나오는 방법은 현재 클릭한 번호가 이전 번호보다 클경우 next 버튼을 누른 슬라이딩효과 그 반대인 경우 prev 버튼을 누른 효과가 나오도록 한다. 보너스 현재 화면에 보이고 있는 div에 맞는 버튼 번호의 색상이 체크 되도록 한다. 이때 현재 체크된 번호만 활성화 되어있고, 나머지는 비활성화 상태로 만든다.
